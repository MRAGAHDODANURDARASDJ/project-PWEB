Viral Kereta Api Melintas Mepet Permukiman, Panik Gak Tuh?

Heri Susanto
detikTravel
Share 0 Tweet 0 Share 0 Whatsapp 0 Telegram 0 0 Komentar
Yogyakarta -

Sebuah video yang memperlihatkan kereta api melintas mepet permukiman viral di media sosial. Diketahui lokasinya di Jogja. Kok bisa sih?

Lokasi kereta api melintas mepet permukiman itu tepatnya ada di RT 37/RW 10, Kampung Pengok, Kelurahan Demangan, Kemantren Gondokusuman, Kota Jogja. Ada sekitar 30 rumah yang berdekatan dengan perlintasan kereta api tersebut.

Kereta api yang melintasi rute ini biasanya menuju Balai Yasa atau bengkel kereta api di Jogja. Kereta yang melintas biasanya berangkat dari Stasiun Lempuyangan.

Baca juga: Akhirnya Mudik Juga, Ini 5 Tips Mudik Sambil Liburan, Biar Nggak Zonk!
Lokasi permukiman di Jogja yang dilintasi kereta api. Foto: Heri Susanto/detikJateng

Ketua RW 10 Zulnasri menjelaskan warga di sekitar rel sudah biasa dengan lalu lalang kereta itu. Alih-alih panik, warga justru menikmati momen kereta melintas sebagai hiburan.

"Warga malah menikmati. Kalau terganggu sudah terbiasa ya. Karena selatan itu rel ganda lebih sering," kata Zulnasri.

Zul mengatakan ada jarak sekitar dua meter antara permukiman warga dengan rel. Warga pun sudah tahu jika rel tersebut bisa kapan saja dilewati kereta, sehingga tak mungkin menaruh barang di atas rel.

"Sudah tahu, tidak ada yang parkir kendaraan di atasnya," katanya.

Sementara itu, Kepala Humas PT KAI Daerah Operasional (Daop) 6 Jogja Supriyanto menyebut rel kereta api yang ada di Jalan Mutiara itu memang sengaja tak dibuatkan perlintasan. Alasannya karena kawasan tersebut area khusus. Selain itu, laju kereta pun cenderung lebih lambat dari kereta pada umumnya. Di samping itu, tak selalu ada kereta yang lewat sana, durasinya bisa dua bulan dua kali saja.
